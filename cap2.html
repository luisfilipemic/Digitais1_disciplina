Números binários

Números binários é um sistema de numeração com peso 2, diferente dos decimais. ou seja, cada digito possui seu respectivo peso \(2^n\).

  • Exemplo:

Fórmula geral:

\[Qtds de numeros para bits = 2^n-1\]

Por que -1?

Por que existe o zero sendo contado.

Estrutura de pesos.

Em um binario cada bit possui respectivo peso. O bit mais esquerda é considerado o menos sisgnificativo (LSB), e a direita o mais significativo(MSB);

Numeros binarios com virgula:

\[2^{n–1}... 2^{3} 2^{2} 2^{1} 2^{0}, 2^{–1} 2^{–2}... 2^{–n}\]

Onde a virgula é denotada como virgula binária;

  • Exemplo 2-3: Converta o número binário inteiro 1101101 para decimal:

teremos:

\[(2*10^0) + (2*10^2) + (2*10^3) + (2*10^5) + (2*10^6) = 109 \]

  • Exemplo 2-4: Converta o número binário fracionário 0,1011 para decimal.

teremos: \[(2*10^{-1}) + (2*10^{-3}) + (2*10^{-4}) = 0,6875\]

COmplementos de 1 e de 2 de numeros binarios:

  • Complemento de 1:

O complemento de 1 de um número binário é determinado trocando-se todos os 1s por 0s e todos os 0s por 1s.

  • Complementow de 2:

\[Complemento de 2 = Complemento de 1 + 1No LSB\]

Lembre-se que a soma é somente no complemento de 1, esqueça o numero definido inicialmente.

Números siinalizados:

  • Bit de sinal:

0 - positivo

1 - negativo

FOrma sinal magnitude

Na forma sinal-magnitude, um número negativo tem os mesmos bits de magnitude co mo o número positivo correspondente mas o bit de sinal é 1 em vez de zero.

Exemplo:

+25 = -25

a magnitude será igual, porém alteramos somente o bit de sinal de 0 para 1;

Complemento de 1:

COmplemento de 1 para numeros positivos:

  • igualmente o sinal magnitude positivo;

COmplemento de 1 para numeros negativos:

  • complementow de 1 para o sinal magnitude.

ex:

-25 = comp1(+25)

+25 = sinalmag(+25)

Complemento de 2:

COmplemento de 2 para numeros positivos:

  • igualmente o sinal magnitude positivo;

COmplemento de 2 para numeros negativos:

  • complementow de 2 para o sinal magnitude.

ex:

-25 = comp2(+25)

+25 = sinalmag(+25)

O valor decimal de numeros sinalizados(processo inverso):

  • SInal magnitude:

no sinal magnitude basta somar os pesos relativos aos bits 1s;

o bit de sinal informará se é positivo ou negativo;

  • Complemento de 1:

Positivos:

Para os positivos basta somar os respectivos pesos 1s

Negastivos: Os valores decimais de números negativos são determinados atribuindo um valor negativo ao peso do bit de sinal, somando os pesos relativos aos bits 1s e somando 1 ao resultado

  • COmplemento de dois:

POsitivos:

Basta somar os respectivos pesos dos bits 1s

negativos:

basta somar o peso dos bits 1s, sendo que o bit sinalizado terá peso negativo na soma;

Observa-se que o complemento de 2 é mais fácil;
